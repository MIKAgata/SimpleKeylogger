### Tujuan Pembelajaran
Proyek ini digunakan untuk memahami:

Bagaimana sistem operasi menangani input keyboard

Cara kerja event listener dalam Python

Konsep logging dan buffer management

Risiko keamanan dari aplikasi yang memonitor input

Simulasi metode data exfiltration untuk tujuan defensive analysis

### Fitur
1.Mendeteksi segala inpu keyboard
2.Dapat berjalan dilatar belakang tanpa disadari
### Etika dan Legalitas
