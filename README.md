# Simple Keylogger (Educational Purpose)

## Deskripsi
**Simple Keylogger** adalah proyek Python sederhana yang dibuat **khusus untuk tujuan edukasi**.  
Program ini mencatat input keyboard beserta timestamp dan menyimpannya ke dalam file teks.

Proyek ini ditujukan untuk:
- Memahami cara kerja **keyboard event listener**
- Mempelajari **input handling** di Python
- Menambah wawasan dasar tentang **bagaimana keylogger bekerja** dari sisi teknis (defensive learning)


---

## Fitur
- Mencatat setiap tombol yang ditekan
- Menyimpan log ke file teks (`keylog.txt`)
- Menyertakan timestamp (`HH:MM:SS`)
- Mendeteksi tombol khusus seperti:
  - SPACE
  - ENTER
  - TAB
  - BACKSPACE
  - SHIFT, CTRL, ALT
- Output real-time ke terminal

---

## Teknologi
- Python 3
- Library `pynput`
- Modul `datetime`
